Textee is a Go package that takes raw text and extracts every 1-, 2-, and 3-word substring, counts how often each appears, and optionally computes six different gematria cipher scores for each substring. It is built for concurrent processing and is safe for use in parallel workloads.

## How it works

<Steps>
  <Step title="Install">
    Add Textee to your Go module.
    ```bash
    go get -u github.com/andreimerlescu/textee
    ```
  </Step>
  <Step title="Parse text">
    Call `NewTextee` with one or more strings. The library splits the input into sentences, then generates every 1-, 2-, and 3-word n-gram and counts occurrences atomically across goroutines.
    ```go
    tt, err := textee.NewTextee("All right, let's move from this point.")
    ```
  </Step>
  <Step title="Read results">
    Iterate over `tt.Substrings` or call `tt.SortedSubstrings()` to get results ordered from most to least frequent.
    ```go
    for _, sq := range tt.SortedSubstrings() {
        fmt.Printf("%s: %d\n", sq.Substring, sq.Quantity)
    }
    ```
  </Step>
  <Step title="Add gematria (optional)">
    Call `tt.CalculateGematria()` to enrich each substring with English, Jewish, Simple, Mystery, Majestic, and Eights cipher scores.
    ```go
    tt, err = tt.CalculateGematria()
    fmt.Println(tt) // prints substrings with all six scores
    ```
  </Step>
</Steps>
